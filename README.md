# MNIST Streamlit Prototype

손글씨 숫자를 입력하면 ONNX MNIST 모델로 추론하고, 결과를 로컬 파일로 다운로드할 수 있는 멀티페이지 앱입니다.

## 주요 기능
- 입력 캔버스에서 숫자 그리기 (선 굵기 조절)
- 28x28 전처리 이미지 시각화
- 모델 추론 결과 및 레이블별 확률 차트 확인
- 이미지 저장소 페이지에서 탭별 다운로드
  - 이미지 다운로드 (PNG)
  - 추론결과 다운로드 (JSON, 소수점 4자리 반올림)
  - 확률표 다운로드 (CSV)
- 모델 관리 페이지
  - 모델 상태 확인
  - 모델 재다운로드
  - ONNX 세션 캐시 재로드

## 페이지 구성
- 홈: `홈.py`
- 입력 + 전처리: `pages/1_입력_및_전처리.py`
- 모델 추론 결과: `pages/2_모델_추론_결과.py`
- 이미지 저장소(다운로드): `pages/3_이미지_저장소.py`
- 모델 관리: `pages/4_모델_관리.py`

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run 홈.py
```

## 참고
- `mnist-12.onnx` 파일이 프로젝트에 포함되어 있으면 해당 모델을 사용합니다.
- 모델 파일이 없으면 `src/mnist_service.py`의 다운로드 로직으로 자동 다운로드를 시도합니다.
